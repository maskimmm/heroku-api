A prototype API for Calculating Expected Energy.

API Name Description API Route Inputs Output
Return Of Investment - /api/v1/return-of-investment {GMT: float, Lat:float, Long:float, ColTilt:float, AzimuthCol:float, RLength:float, RWidth:float, PLN:float} Json of {Year:{Tahun, YearlyIncome, OMCost, IncomeYearN, ROI}} (each in float)
Net Present Value - /api/v1/net-present-value {GMT: float, Lat:float, Long:float, ColTilt:float, AzimuthCol:float, RLength:float, RWidth:float, PLN:float} Json of {Year:{Tahun, CapitalCost, YearlyIncome, OMCost, Rt, NPV}} (each in float)
PV Output Daily and Yearly, Expected Energy Daily and Yearly - /api/v1/all-output {GMT: float, Lat:float, Long:float, ColTilt:float, AzimuthCol:float, RLength:float, RWidth:float, PLN:float} Json of [pvYearly, pvDaily, eeDaily, eeYearly] (each in float)
Expected Energy SVI Yearly Used to Calculate Yearly Expected Energy after SVI /api/v1/energy-svi-yearly {GMT: float, Lat:float, Long:float, ColTilt:float, AzimuthCol:float, RLength:float, RWidth:float, PLN:float} Expected Energy SVI Yearly (float)
Expected Energy SVI Daily Used to Calculate Daily Expected Energy after SVI /api/v1/energy-svi-daily {GMT: float, Lat:float, Long:float, ColTilt:float, AzimuthCol:float, RLength:float, RWidth:float, PLN:float} Expected Energy SVI Daily (float)
PV Output Daily Used to Calculate Daily PV Output /api/v1/pv-output-daily {GMT: float, Lat:float, Long:float, ColTilt:float, AzimuthCol:float, RLength:float, RWidth:float, PLN:float} PV Output Daily (float)
PV Output Yearly Used to Calculate Yearly PV Output /api/v1/pv-output-yearly {GMT: float, Lat:float, Long:float, ColTilt:float, AzimuthCol:float, RLength:float, RWidth:float, PLN:float} PV Output Yearly (float)

How to use this API Using Postman:

  1. Set Method as 'POST' Methods
  2. Insert Link, start with api-calculation.herokuapp.com + API Route above. (Example: To use Return Of Investment Feature, type this link api-calculation.herokuapp.com/api/v1/return-of-investment)
  3. Select Body > raw
  4. In raw, put all 'Inputs' above with formats: {"Key": Value} . (Example: {"GMT":7, "Lat":-7.2855, "Long":112.796, "ColTilt":15, "AzimuthCol":-90, "RLength":4, "RWidth":2, "PLN":1300})
